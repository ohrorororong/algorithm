## 1. 순열 (Permutation)
* ### 순열이란? 
> 서로 다른 n개의 원소에서 r개를 중복없이 **순서에 상관있게** 선택하는 혹은 나열하는 것을 순열(permutation)이라고 한다. 
>
> [나무위키](https://namu.wiki/w/%EC%88%9C%EC%97%B4)

### 문제 풀이
1. PermutationPractice1 : 재귀함수로 순열 구현
2. PermutationPractice2 : swap 함수로 순열 구현
